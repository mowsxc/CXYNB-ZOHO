# Agent 指令

## 思考过程语言
所有内部思考、推理、分析过程必须使用简体中文。技术术语可保留英文，句子的主体必须是中文。

## 每次完成任务前
1. 读取 `.monkeycode/MEMORY.md`，逐条执行其中的规则
2. 读取本文件，遵守以下工作流程

## 工作流程
1. 当审计发现问题时，全部修复后再跑第二轮审计验证，确认零问题后再报告
2. 报告时只给结论，不列出中间过程
3. 每次改动后必须执行 `git add + commit + push`

## 代码质量标准
- 每次提交前全符号/命名/引用一致性检查
- 检查函数调用链完整性
- 检查 DOM id/class 引用与 HTML 定义一致
- 检查变量命名新旧一致性
- 检查所有路径覆盖（首次渲染/增量渲染/离线/在线）
- 提交前运行 lint/语法验证
